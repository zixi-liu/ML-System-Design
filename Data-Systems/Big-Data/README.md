## 大数据经典论文解读

**大数据的核心概念**
- 能够伸缩到一千台服务器以上的分布式数据处理集群的技术。
- 这个上千个节点的集群，是采用廉价的 PC 架构搭建起来的。
- “把数据中心当作是一台计算机”（Datacenter as a Computer）。

GFS、MapReduce 和 Bigtable 完成了“存储”“计算”“实时服务”这三个核心架构的设计。

Spark 通过把数据放在内存而不是硬盘里（数据缓存在内存），大大提升了分布式数据计算性能。
